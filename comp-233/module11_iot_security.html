Module 11 – Database Security (IoT Light Sensor)

Agenda

  • SQL Injection
  • Encryption
  • Access Control
  • Distributed DB Security
  • Best Practices

SQL Injection Example

SELECT * FROM sensor_readings
WHERE room_id = 'lab1';

Attack:

' OR '1'='1

Result: All sensor data exposed

SQL Injection Fix

SELECT * FROM sensor_readings
WHERE room_id = ?;

stmt = conn.prepare(
"SELECT * FROM sensor_readings WHERE room_id = ?"
);
stmt.execute(input);

Encryption Example

pgp_sym_encrypt('abc123','key')

INSERT INTO device_credentials(key)
VALUES (
pgp_sym_encrypt('abc123','key')
);

Decryption

SELECT pgp_sym_decrypt(key,'key')
FROM device_credentials;

Access Control

GRANT SELECT ON sensor_readings TO user1;
GRANT INSERT, UPDATE ON sensor_readings TO manager;
REVOKE DELETE ON sensor_readings FROM user1;

Distributed DB (Separation)

Public DB:
sensor_readings(light_level)

Private DB:
device_credentials(key)
users(password)

Best Practices

  • Use prepared statements
  • Encrypt sensitive data
  • Apply least privilege
  • Log and audit all actions
  • Monitor anomalies
