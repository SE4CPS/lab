1. Flexbox Card Layout

These cards use display: flex, flex-wrap, and flex: 1 1 250px. That means they take steady steps across the row, then stack when the ground gets narrow.

Flexible Width

Each card can grow and shrink while respecting a minimum practical width.

Wrapping

When there is not enough horizontal space, cards move to the next line instead of becoming too cramped.

Clean Structure

This older, dependable pattern is useful for features, pricing boxes, and content summaries.

2. CSS Grid Example

This section uses repeat(auto-fit, minmax(220px, 1fr)). The columns adjust by themselves, which is often a sound way to make progress without hard-coding every size.

Grid Item 1

Auto-fit keeps the layout balanced.

Grid Item 2

Minmax protects readability on small screens.

Grid Item 3

Items rearrange as the available width changes.

Grid Item 4

Useful for galleries, cards, and dashboards.

3. Responsive Two-Column Layout

On larger screens, the sidebar sits to the left and the content to the right. On smaller screens, the layout changes direction and stacks into one column.

Sidebar

  • Navigation
  • Quick links
  • Announcements
  • Categories

Main Content Area

This area has more space on wide screens. Below 900px, the layout becomes a single column. That is often the safer path, like changing direction when the ground becomes wet and narrow.

In practical work, this pattern fits blogs, admin panels, class pages, and documentation sites.

4. Responsive Images

Images below use max-width: 100% and height: auto. They stay inside their container rather than spilling out.

5. Core CSS Pattern Used Here

.container {
  width: 90%;
  max-width: 1100px;
  margin: 0 auto;
}

.cards {
  display: flex;
  flex-wrap: wrap;
  gap: 1rem;
}

.card {
  flex: 1 1 250px;
}

.grid-demo {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(220px, 1fr));
  gap: 1rem;
}

.layout {
  display: grid;
  grid-template-columns: 260px 1fr;
}

@media (max-width: 900px) {
  .layout {
    grid-template-columns: 1fr;
  }
}

Responsive design works best when the layout adjusts in measured steps
